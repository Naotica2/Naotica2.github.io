Project Overview

Website Project 2 bertujuan untuk memudahkan para player di server AbsorbedSMP Pada Project 1, memudahkan dalam hal membeli Rank pada server AbsorbedSMP, Terdapat pilihan paket Rank beserta harga dan fitur agar Player tidak kesusahan Memilih rank, Website ini juga bertujuan supaya Player mengetahui Rules server dan Mengetahui link Vote server AbsorbedSMP. Website ini juga mempunyai fitur Beli Rank masukan Gamertag lalu Rank tersebut masuk ke akun player saat di in game, Ringkasnya ini memudahkan player dalam berbagai hal.

Tidak sampai di situ saja, Website ini memiliki fitur News atau pemberitahuan pembaruan baru pada AbsorbedSMP. Website ini saya kembangkan sendiri tanpa ada bantuan lain, Meski memiliki waktu banyak untuk membuatnya tapi itu sangat Worth It.
